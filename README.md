# Zero Trust Architecture Based Banking Application

A cloud-based banking web application implementing Zero Trust Architecture (ZTA). 
The system ensures secure access using JWT authentication, continuous verification, 
device context validation, and least-privilege transaction control.

## Features
- Secure login with JWT authentication
- Zero Trust Architecture implementation
- Protected banking APIs
- Device and request validation
- Cloud deployable architecture

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express
- Security: JWT, Zero Trust principles
- Deployment: Cloud platform
